# **App Name**: RouteWise

## Core Features:

- Map Display: Display an interactive map using Mapbox API showing customer booking locations.
- Job Listing: Display a list of jobs with customer name, address, and scheduled time.
- Route Calculation: Calculate and display the optimized travel routes between jobs using Mapbox Directions API.
- Route Optimization: Employ an LLM as a tool for optimally reordering stops by receiving estimated arrival times.
- Distance and Time Estimation: Show the total route distance and estimated total travel time.
- Route Recalculation: Implement a button to recalculate and optimize the route, updating the map and ETA.
- Responsive Layout: Ensure the layout is responsive for different screen sizes.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke trust and reliability, reflecting the optimization aspect of the app.
- Background color: Light grey (#F0F2F5) to provide a clean and neutral backdrop.
- Accent color: Vivid orange (#FF5722) to highlight key actions and information, like the recalculate button.
- Body and headline font: 'Inter', a sans-serif for a clean and modern dashboard appearance.
- Note: currently only Google Fonts are supported.
- Use clear and consistent icons from a library like FontAwesome to represent different job types or actions.
- Use a split-screen layout with the job list on the left and the map on the right for balanced information presentation.
- Implement smooth transitions and map marker animations to enhance the user experience.
